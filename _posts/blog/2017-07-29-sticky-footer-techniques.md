There are a number of methods for forcing a footer to the bottom of the page no matter the height of the content in your page.

## First, Establish The Properties of Your Site

## Simplest Method

{% highlight html %}
<div class="wrap">
  <body></body>
  <footer></footer>
</div>
{% endhighlight %}

{% highlight css %}
.wrap {
  min-hight: calc(100vh - 80px);
}
{% endhighlight %}
